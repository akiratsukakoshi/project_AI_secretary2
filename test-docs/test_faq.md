# よくある質問

## Q: RAGシステムとは何ですか？

A: RAG（検索拡張生成）システムは、外部の知識ベースを活用して生成AIの応答を強化するアプローチです。検索エンジンのように情報を取得し、AIの応答生成に活用します。

## Q: なぜRAGシステムが必要なのですか？

A: 大規模言語モデルは膨大な知識を持っていますが、トレーニングデータ以降の情報や特定のドメイン知識には弱い面があります。RAGを使うことで、最新情報や特定の組織知識をAIの応答に反映させることができます。
